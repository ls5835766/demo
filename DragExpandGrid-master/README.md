# DragExpandGrid

这是一个多功能的扩展GridView

1. 可展开，可拖动，可排序，可删除。
2. 固定更多按钮。
3. 展开合并支持动画。
4. 支持箭头图标移动。


数据的处理和显示使用Bean。


来自于[500彩票Andorid客户端](http://500.com )首页功能。
